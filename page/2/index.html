mac升级自带的PHP版本

php –version →查看当前版本
curl -s http://php-osx.liip.ch/install.sh | bash -s 7.0 →安装指定版本

如果当前本机有旧版本，那么新版本不会覆盖原版本，需要手动替换掉旧版本，如下：

sudo vi ~/.bashrc → 修改系统环境变量
增加下面语句
export PATH=”/usr/local/bin:/usr/local/sbin:$PATH”

OK，结束

正则学习笔记

##符号

^ ：脱字符 表示求反的概念&匹配开头的字符。
例如[^abc]：表示除了a、b、c之外的任意字符。

[ ]：表示可以是中括号中的任何一个字符。
例如[abc]：表示匹配a、b、c中的任何一个；

{ }：表示匹配数量。
例如：/b{m,n}/表示匹配字符串中b出现最少m次，最多n次的情况。

| ：管道符，表示其中任何之一。
例如：（1|a|9）表示可以使1、a、9三个字符串中的任何一个。
